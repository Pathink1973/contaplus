Como usar o ContaPlus

Adicionar uma Nova Conta

Use o formulário para adicionar uma nova conta com:

  • Nome do serviço ou conta
  • Valor em euros
  • Data de vencimento

Gerenciar Contas

Para cada conta você pode:

  • Editar os detalhes usando o botão de lápis
  • Excluir a conta usando o botão de lixeira
  • Ver alertas de vencimento próximo (5 dias)

Recursos

  • Tema claro/escuro (botão sol/lua)
  • Resumo total das contas
  • Alertas de vencimento
  • Organização por data
